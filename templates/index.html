HealthGuard AI

AI-Based Self-Care Recommendation & Health Monitoring System

HealthGuard AI is a Flask web application that uses Machine Learning to predict possible diseases from symptoms, estimate health risk & provide personalised self-care recommendations.

Core Technologies
Python Flask Random Forest Pandas Bootstrap 5 HTML5 CSS3 JavaScript

Key Features

AI Disease Prediction

Uses a Random Forest model to predict likely diseases from selected symptoms.

Health Risk Analysis

Estimates the patient's risk level and provides personalised recommendations.

Secure Processing

All predictions are processed locally without sending personal data externally.
